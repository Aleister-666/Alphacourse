# Alphacourse

Sistema tutorial para el aprendizaje del uso educactivo de la plataforma moodle para profesores. Creado con Ruby on Rails + Turbo + Stimulus

### Caracteristicas
* Administrador:
	- Crear Secciones
	- Crear Cursos
	- Crear Pruebas
	- Inspeccionar y Eliminar Usuarios(Admin y estudiante)
	- Control Total
* Usuario Normal:
	- Crear cuenta de Estudiante
	- Incribirse a los cursos
	- Presentar Pruebas
	- Completar modulos de pruebas
	- COmpletar Cursos

## Requisitos
* Ruby version: 3.1+
* Postgresql: 15+
* libvips
* Gemas:
	- haml
	- annotate
	- devise
	- sassc-rails
	- rails-i18n
	- acts_as_list
	- requestjs-rails
	- font-awesome-sass
	- image_processing
	- Pagy
